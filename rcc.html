What is RCC?

RCC (Really Cool C-generator?) is an interactive AWK script that helps you build C programs line by line through a friendly menu system. No more staring at blank files wondering where to start! It was originally made in rc shell hence the name.

Features

  • Build C code interactively with simple commands
  • No need to remember exact syntax
  • Friendly ASCII art and emoticons
  • Save and load your creations
  • Built in/for 9front
  • Perfect for learning or quick prototyping

Quick Start

$ awk -f rcc.awk

Philosophy

Learning to code shouldn't feel like climbing a mountain. Sometimes you just want to build something without worrying about every semicolon and bracket. RCC lets you focus on the logic while it handles the syntax. I just wanted to share a quirky toy!
